# DoaZap — Landing Page

Landing page estática pronta para publicação no **GitHub Pages**.

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `seu-usuario.github.io` ou `doazap`)
2. Faça upload dos arquivos deste projeto para o repositório
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `Deploy from a branch → main → / (root)`
5. Clique em **Save** — em poucos minutos o site estará no ar!

## 📁 Estrutura dos arquivos

```
/
├── index.html       ← Página principal (arquivo único, tudo embutido)
└── README.md        ← Este arquivo
```
